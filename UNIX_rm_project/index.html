The rm (remove) command in Unix

The rm command on the unix command line is used to remove files.

The rm command appeared in Version 1 AT&T Unix.

Options Associated with the rm command:

-d Removes directories and other types of files.
-i Removes files after prompting yes or no.
-f Removes files without prompting.
-P Overwrites files before deleting them.
-r or -R Attempts to remove the file hierarchy rooted in each file agrument. This option implies the -d (directories) option.
-v Verbose when deleting files; each file is shown as they are removed.
-W Attempts to recover (undelete) the named file.
