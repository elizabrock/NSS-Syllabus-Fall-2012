Prompting When Deleting

Options -d and -i.

There is no recycle bin or trash can when deleting files and directories on the command line.

Sometimes it is possible to retrieve files that were deleted from the command line, but there is no guarantee.

To insure that an incorrect file, set of files or directory, the option -i can be used so that Unix will prompt the user to confirm the deletion of each file (yes or no). The command -d can be used to cancel the prompting.

An Alias

Some users will create an alias, which will make prompting the default:
   alias rm="rm -i"
   rm() { /bin/rm -i "$a" ; }
