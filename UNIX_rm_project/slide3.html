Removing Directories

Options -d and -R (or -r).

Option -d, (for "directory") will delete an entire directory, when that directory is passed through as an argument. This option will attempt to remove directories as well as other types of files.

Option -R (also -r), for "recursive" will attempt remove the file hierarchy rooted in each file argument. The -R option implies the -d option. If the -i option is specified, the user is prompted for confirmation before each directory's contents are processed (as well as before the attempt is made to remove the directory). If the user does not respond affirmatively, the file hierarchy rooted in that directory is skipped.
